# Corso: Metodi Computazionali per Modelli Stocastici

[![License: CC BY 4.0](https://img.shields.io/badge/License-CC%20BY%204.0-lightgrey.svg)](https://creativecommons.org/licenses/by/4.0/)
[![License: MIT](https://img.shields.io/badge/License-MIT-blue.svg)](https://opensource.org/licenses/MIT)

Materiali didattici del corso di **Metodi Computazionali per Modelli Stocastici**, a cura di **Antonio Scala**.  
Il corso copre concetti teorici e metodi numerici per la simulazione e lanalisi di sistemi stocastici, con applicazioni a fisica, biologia, finanza ed epidemiologia.

Tutti i materiali sono in formato Markdown compatibile Pandoc, pensati per la conversione in PDF o HTML e per luso didattico interattivo.

---

## Struttura del repository

- `dispense/` Dispense delle lezioni teoriche (file `.md` numerati 0110)  
- `dispense/codice/` Codici Python comuni alle lezioni  
- `dispense/dati/` Dataset di esempio per simulazioni  
- `dispense/immagini/` Figure e grafici utilizzati nei capitoli  
- `dispense/refs/` Riferimenti bibliografici e file `.bib`  
- `laboratori/` Esercitazioni numeriche e notebook  
- `progetti/` Proposte di progetti interdisciplinari  
- `slide/` Slide di supporto alle lezioni  
- `template/` Template Pandoc e LaTeX per esportazione  
- `config.yaml` Impostazioni comuni di compilazione

## Software consigliato

| Strumento | Scopo principale | Comando di installazione (Ubuntu / Debian) |
|------------|------------------|--------------------------------------------|
| **Pandoc  3.1** | Conversione e compilazione Markdown  PDF / LaTeX | `sudo apt install pandoc` |
| **TeX Live Full** | Ambiente LaTeX completo (per esportazione PDF) | `sudo apt install texlive-full` |
| **Python 3.x + pip** | Esercitazioni e simulazioni numeriche | `sudo apt install python3 python3-pip` |
| **Zettlr** | Editor Markdown integrato con Pandoc e BibTeX | `sudo apt install zettlr` |
| **Biber / BibTeX** | Gestione bibliografia accademica | `sudo apt install biber` |
| **Git** | Versionamento e sincronizzazione GitHub | `sudo apt install git` |

Tutti gli strumenti sono multipiattaforma (Linux, macOS, Windows) e compatibili con lambiente di lavoro Pandoc + LaTeX previsto dal corso.
